Attach a framebuffer for a WlBuffer

This tries to import the buffer to gbm and attach a framebuffer::Handle for the imported [BufferObject].

Returns Ok(None) for unknown buffer types and buffer types that do not support attaching a framebuffer (e.g. shm-buffers)
