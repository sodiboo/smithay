Handler utilities for the various selection protocols.

Wayland originally only had a singe way to deal with the user clipboard with the help of wl_data_device protocol. However the demand for primary selection as well as writing clipboard managers made it into developing protocol extension for the said cases.

The following selection providers are supported:

  • The data_device module to work with the traditional clipboard selection.
  • The primary_selection module to work with the primary selection.
  • The wlr_data_control module to hook data control into clipboard and primary selection
